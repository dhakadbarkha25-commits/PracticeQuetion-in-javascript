# PracticeQuetion-in-javascript



1. Can we connect JavaScript directly with database?
a) Yes
b) No
c) Sometimes
d) Some databases
<details> <summary>View Answer</summary>

Answer: b) No

Reason (English):
Frontend JavaScript cannot directly connect to database.

Reason (Hindi):
Browser wali JS direct database se connect nahi kar sakti kyunki credentials leak ho jate hain aur security risk hota hai.

Comment:
Hamesha backend (Node.js, PHP) use hota hai.
